# Vue 3 + TypeScript + Vite

內容:
```
網頁包含登入系統，使用者登入後會通過Line推播通知使用者狀態。
通過Google Map API 在網頁上顯示，並可以選擇座標。
在地圖上建立相簿、敘述、地點和相關照片，並將資料儲存至資料庫。
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
